Trailblaze

Trailblaze is a web application meant for runners interested in exploring hiking! The app allows users link their Strava account and use their past runs to generate hikes near them using ChatGPT. Once the hikes are generated, the user will have basic information about the top 10 hikes near them including a link to the hike AllTrails for more details. TrailBlaze was developed as a Python Flask Application, using HTML and CSS for the front end and Python for the backend. The app also uses SQLite to store user data.

Future Improvements

In the future, I would like to add more features to the app such as the ability to save hikes, share hikes with friends, add additional information about the hikes, and use other Strava activities to generate the hikes. I would also like to improve the user interface and make the app more mobile friendly.

LearnPrivacy

LearnPrivacy is a web application that allows users to learn about more about privacy and privacy computing techniques such as K-Anonymity and Differential Privacy. The app provides information about privacy and how the techniques work. Each technique has a dedicated page with information about the technique and a hands on experience where the user can see the technique in action to enhance the learning experience. LearnPrivacy was developed as a Python Flask Application, using HTML and CSS for the front end and Python for the backend. The app also uses Boostrap and Python libraries such as Pandas, Numpy, pyCanon, and python-DP.

Future Improvements

In the future, I would like to add more privacy techniques to the app, improve the user interface, and make the app more mobile friendly. I would also like to add a "developer mode" where users can see the code behind the techniques and how they are implemented.
